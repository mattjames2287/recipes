# Recipe Book

GitHub Pages recipe site with:
- kitchen background image
- cookbook overlay on the counter
- flip-style recipe card
- Add Recipe form
- starter recipes from `data/recipes.json`
- new recipes saved in browser local storage

## Files
- `index.html`
- `style.css`
- `app.js`
- `data/recipes.json`
- `assets/kitchen-bg.png`

## Add your own recipes permanently
Edit `data/recipes.json` and add recipe objects.

## Add your own food photos
Upload them into your repo and use the image path in your recipes, for example:
`assets/images/lasagna.jpg`

## Note
Recipes added with the on-page form save only in that browser/device unless you also copy them into `data/recipes.json`.
