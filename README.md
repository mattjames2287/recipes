# Bari Family Cook Book

This is a GitHub Pages recipe book with a kitchen-scene background and a cookbook overlay.

## What changed
- Title updated to **Bari Family Cook Book**
- Removed the old Form Tip box
- Added category pills in the top filter row
- Added category pill buttons inside the Add Recipe form
- Warped the cookbook overlay more so it sits over the printed book in the background
- Added another highland cow to the kitchen background
- Kept the site mobile-friendly so it opens well on a phone

## Important shared-use note
This site works on your wife's phone for viewing and using the form.

Recipes added with the form are saved in that specific browser using local storage, so:
- recipes added on your phone stay on your phone
- recipes added on her phone stay on her phone

To make recipes shared for both of you, also copy them into:

`data/recipes.json`

That file is part of the site and will be the shared version once pushed to GitHub Pages.

## Publish
Upload the `recipe_site` folder contents to your GitHub Pages repo root, or copy the files into an existing repo and enable Pages.
